Project Description:
This project implements and evaluates Part-of-Speech (POS) tagging models using the English Penn Treebank dataset. Two models were developed and compared: Logistic Regression and Bidirectional LSTM (BiLSTM).

Requirements:

* Python 3.x
* pandas
* numpy
* matplotlib
* seaborn
* scikit-learn
* torch

How to Run:

1. Upload the training and development datasets.
2. Open the Jupyter Notebook.
3. Run all cells from top to bottom.
4. Results, figures, and evaluation metrics will be generated automatically.

Files Included:

* Project Notebook (.ipynb)
* PDF Export
* README.txt

Main Result:
The BiLSTM model achieved 96.14% accuracy and 87.36% Macro F1 on the development set.
